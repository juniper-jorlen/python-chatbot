# Simple Python Chatbot

## Description
This is a simple chatbot built in Python. It responds to basic user inputs such as greetings, asking its name, and help commands. The chatbot runs in the terminal and continues the conversation until the user types `bye`.

## Features
- Responds to greetings
- Answers simple questions
- Exits when the user types `bye`
- Easy to modify and extend

## Technologies Used
- Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/yourusername/python-chatbot.git
```

2. Open the project folder:

```bash
cd python-chatbot
```

3. Run the chatbot:

```bash
python chatbot.py
```

## Example

```
You: hello
Bot: Hello! How are you?

You: what is your name
Bot: I'm a simple Python chatbot.

You: bye
Bot: Goodbye!
```
